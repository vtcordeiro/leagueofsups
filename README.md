**League of Sup**

**Descrição**

Este projeto foi criado na Imersão Dev da Alura com parceria com o Google Gemini e é um buscador de campeões de League of Legends, com foco em campeões de suporte. O usuário pode pesquisar por nome ou estilo de jogo e obter informações detalhadas sobre cada campeão.

**Tecnologias Utilizadas**
* HTML
* CSS
* JavaScript

**Como Usar**

1. **Clone o repositório:**
   ```bash
   git clone https://seu-repositorio.git
   ```
2. **Abra o projeto:**
   Abra o arquivo `index.html` em seu navegador.

**Estrutura do Projeto**

* **index.html:** Página principal do projeto.
* **style.css:** Arquivo de estilos CSS.
* **app.js:** Lógica JavaScript da aplicação.
* **dados.js:** Arquivo contendo os dados dos campeões.
